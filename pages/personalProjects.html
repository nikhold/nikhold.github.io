This Portfolio

Utilizing some basic web development and design skills, I created this portfolio completely from scratch. I wrote it in HTML and CSS in VS Codium (base component of VS Code), hosted it on GitHub Pages, and put it together in order to showcase my projects. Since I already had some basic knowledge writing HTML and CSS, this project was somewhat simple and did not take very long. Some help was required for more advanced elements, so I give a big thanks to W3Schools for all the help it could provide. I hope you enjoy the website and that you find it easy to navigate.

My 3D Printer

This is a project that really got me into hardware specifics and applications. I wanted to understand what I could improve to make my prints turn out better, and it ended up teaching me a lot about hardware's physical interaction with the world around it and the importance of a proper environment. I have taken a basic Ender-3 and transformed it into something my own that does the job efficiently. I have added new circuit boards, wires, motors, hotends, etc. in order to perfect my prints.
